About Daniel's Capstone Project

The WeatherX Service...

a cutting edge Weather Service and an App.

Using Several API sources: MapBox, DarkSky, TimeZoneDB, ArcGIS and Google Earth...

to deliver and visualize current and forecast weather for any place on Earth.

WeatherX GitHub Repo

Latest Technologies

Thanks to Team TreeHouse, a great place for anyone to get started in Software Development.

This project uses Node.js and Express.Js for https server and routing, Mongoose for db backend, React.Js and Babel.js for front-end UI/UX

And it's all javascript es2015, web-app best-practices, TSL2 security and html5 web-standards compliant!
