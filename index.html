About Daniel's Capstone Project

WeatherX:

WeatherX is combination Server and Front-end web app.

It uses 2 API sources, TomTom and DarkSky, to get weather data for any place on Earth.

The Server...

provides data to the front-end. It is a node/express.js server, with a SQLite/Sequelize database, accesses a external REST API using async method.

The frontend web app....

is built mobile-first, using react.js web application framework

uses React-Router and Server-Side-Rendering
